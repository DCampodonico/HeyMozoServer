Servidor Api-Rest para la aplicación HeyMozo.

Introducción
----------

Este proyecto es un servidor de pruebas backend para proveer una API RESTful a las aplicaciones clientes utilizando Java y MongoDB.

Instalación
----------

1. Instalar y ejecutar una base de datos MongoDB.

2. Ejecutar el programa Scripts, el cual poblará la base de datos de datos de prueba. (Opcional)

3. Ejecutar el programa Main, el cual levantará el servidor en el puerto 3000.
